# Mr Hefner — Hermes Power Deck

A rich, self-contained HTML slide deck demonstrating what Hugh / “Mr Hefner” can do now that Hermes is connected with OpenAI, OpenRouter, Gemini, 1Password CLI, Vercel CLI, GitHub CLI, Agent Zero, and AntiGravity.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This repo is designed for Vercel as a static Vite site.
